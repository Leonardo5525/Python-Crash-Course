# **List**
## Accessing Elements in a List
```py
bicycles = ['trek', 'cannondale', 'redline', 'specialized']
print(bicycles[0])
-> trek
```

## Returning last value on the list or -2 (before the last) -3 (befora -2) and continue

```py
bicycles = ['trek', 'cannondale', 'redline', 'specialized']
print(bicycles[-1])
-> specialized
```
